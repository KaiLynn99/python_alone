## 흐름

1. 각 문자를 순회하며 오리의 울음 상태를 추적한다.
2. `left`: 현재 쉬고 있는, 즉 `quack`을 완성한 오리 수
3. `count[0~3]`: `q`, `u`, `a`, `c`까지 진행 중인 오리 수
4. `q`를 만나면 쉬는 오리가 있을 경우 재활용하고, 없으면 새 오리를 추가한다.
5. `u`, `a`, `c`, `k`를 만나면 이전 단계의 오리가 있어야 한다.
6. `k`를 만나면 오리 한 마리가 울음을 완성하고 다시 `left`로 돌아간다.
7. 순회가 끝난 뒤 진행 중인 오리가 남아 있다면 `-1`을 출력한다.

## 코드

```python
string = input().strip()

# 이곳에 풀이 코드를 작성
print(string)
```
